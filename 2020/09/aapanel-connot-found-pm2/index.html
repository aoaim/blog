aaPanel pm2 提示 command not found

2020-09-14
興趣學習

可能是系統變量出現問題。

  • 為 pm2 添加軟鏈接
ln <span class="hljs-_">-s</span> /www/server/nvm/versions/node/v12.18.3/bin/pm2 /usr/bin/pm2

如出現 /usr/bin/env: node: No such file or directory 説明 node 的系統變量出現問題。 - 為 node 添加軟鏈接

ln <span class="hljs-_">-s</span> /www/server/nvm/versions/node/v12.18.3/bin/node /usr/bin/node

最後 pm2 再添加項目時，發現 npm 也報錯，同理添加軟鏈接。 、

ln <span class="hljs-_">-s</span> /www/server/nvm/versions/node/v12.18.3/bin/npm /usr/bin/npm

以上具體路徑務必按照具體情況設置，不可照搬。 如果是通過命令安裝，需要將軟路徑更改爲安裝路徑。 順帶吐槽一下 aaPanel，各種小問題。。。
